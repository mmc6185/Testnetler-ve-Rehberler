## Komutu Çalıştırıp Kurulumu Başlatalım

```
wget -O zeeka.sh https://raw.githubusercontent.com/brsbrc/Testnetler-ve-Rehberler/main/Zeeka-Network/zeeka.sh && chmod +x zeeka.sh && ./zeeka.sh
```
